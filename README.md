# Address Book## Project DescriptionThis is a C++ address book program that manages different types of contacts. The program supports loading contacts from a CSV file, displaying contact information, adding new contacts, editing existing contacts, searching contacts, deleting contacts, and saving contact data.The project uses object-oriented programming with separate classes for contact data and address book management.## Features- Load contacts from `contacts.csv`- Display all contacts- View contact details by ID- Add new contacts- Edit existing contacts- Delete contacts by ID- Search contacts by name, phone number, email, group, or tag- Save contacts back to a CSV file- Support different contact types, including:    - Person    - Business    - Vendor## Project Files- `main.cpp`    Runs the program demos and tests the main address book functions.- `Contact.h` / `Contact.cpp`    Defines and implements the Contact class, including contact information such as name, phone number, email, address, group, tags, and type-specific fields.- `AddressBook.h` / `AddressBook.cpp`    Defines and implements the AddressBook class, including adding, editing, deleting, searching, displaying, loading, and saving contacts.- `contacts.csv`    Stores sample contact data used by the program.- `CMakeLists.txt`    Builds the project using CMake.## How to Run1. Open the project in CLion.2. Make sure `contacts.csv` is in the project folder.3. Build the project.4. Run `main.cpp`.The program will run a series of demos to show that the address book functions work correctly.## NotesThis project demonstrates C++ class design, file input/output, vectors, object management, searching, editing, deleting, and CSV data handling.
