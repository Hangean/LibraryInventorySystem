# Library Inventory System## Project DescriptionThis project is a console-based Library Inventory System written in C++.The program manages different types of library items, including:* Books* Movies* MagazinesThe system stores items using shelves and compartments. Each shelf contains up to 15 compartments, and each compartment can hold one item.The application supports adding, removing, checking out, checking in, printing, swapping, file loading, and accessing items using the `[][]` operator.Example:```cpplibraryInventory[2][4];```This accesses shelf 2, compartment 4.---## Features### Storage ManagementThe program can:* Store items by shelf and compartment location* Store one item per compartment* Access items using `libraryInventory[shelf][compartment]`* Handle invalid shelf and compartment access### Item TypesThe program supports:* Book* Movie* MagazineAll items store:* Name* Description* ID* Checkout status* Borrower name* Due dateBooks also store title, author, and copyright date.Movies also store title, director, and main actors.Magazines also store edition and main article title.### Checkout and CheckinThe program can:* Check out an item by shelf and compartment* Check in an item by shelf and compartment* Check out an item by ID* Check in an item by ID* Store borrower name and due date* Prevent invalid checkout and checkin operations### PrintingThe program can:* Print all items currently checked into storage* Print all checked-out items with borrower name and due date* Display item details using the overloaded `<<` operator### Swapping and RemovingThe program can:* Swap two items between compartments* Prevent swapping if either compartment is empty* Remove an item from a specific shelf and compartment### File LoadingThe program can:* Load library items from `items.txt`* Validate file format* Handle invalid file lines---## Files```textItem.h                Item base class declarationItem.cpp              Item base class File Input FormatEach line in `items.txt` uses `|` as the delimiter.```textBOOK|shelf|compartment|name|description|id|title|author|copyrightDateMOVIE|shelf|compartment|name|description|id|title|director|actor1,actor2,actor3MAGAZINE|shelf|compartment|name|description|id|edition|mainArticleTitle```---## TestingThe `main.cpp` file tests:* Loading items from file* Printing stored items* Accessing items using `[][]`* Adding an item manually* Checking out and checking in items* Printing checked-out items* Swapping items* Removing items* Handling invalid operations---## OOP Concepts UsedThis project demonstrates:* Encapsulation* Inheritance* Polymorphism* Abstract base class* Virtual functions* Operator overloading* Dynamic memory management using `unique_ptr`---## How to RunCompile and run the project using a C++ compiler that supports C++20.Make sure `items.txt` is in the program working directory.In CLion, either copy `items.txt` into the `cmake-build-debug` folder or set the working directory to the project folder.
